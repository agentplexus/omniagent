## [v0.3.0] - 2026-02-22

### Highlights

- Skill system with OpenClaw/ClawHub compatibility
- WASM and Docker sandboxing for secure tool execution

### Added

- `skills/` package with SKILL.md parser supporting YAML frontmatter and metadata ([`72b7281`](https://github.com/agentplexus/omniagent/commit/72b7281))
- Skill discovery from multiple directories with deduplication ([`72b7281`](https://github.com/agentplexus/omniagent/commit/72b7281))
- Requirement checking for skills (bins, env vars) with install hints ([`72b7281`](https://github.com/agentplexus/omniagent/commit/72b7281))
- Skill injection into system prompt with emoji support ([`72b7281`](https://github.com/agentplexus/omniagent/commit/72b7281))
- CLI commands: `skills list`, `skills info`, `skills check` ([`72b7281`](https://github.com/agentplexus/omniagent/commit/72b7281))
- Agent integration with skill loading on startup ([`72b7281`](https://github.com/agentplexus/omniagent/commit/72b7281))
- `sandbox/` package with capability-based permission model ([`f1d3781`](https://github.com/agentplexus/omniagent/commit/f1d3781))
- WASM runtime wrapper using wazero with memory limits and timeout enforcement ([`f1d3781`](https://github.com/agentplexus/omniagent/commit/f1d3781))
- Host functions for sandboxed file, HTTP, and command execution ([`f1d3781`](https://github.com/agentplexus/omniagent/commit/f1d3781))
- Docker sandbox with container isolation and security hardening ([`f1d3781`](https://github.com/agentplexus/omniagent/commit/f1d3781))
- Path validation with symlink resolution for sandbox security ([`f1d3781`](https://github.com/agentplexus/omniagent/commit/f1d3781))
- Command allowlist enforcement for exec operations ([`f1d3781`](https://github.com/agentplexus/omniagent/commit/f1d3781))

### Dependencies

## [v0.2.0] - 2026-02-22
